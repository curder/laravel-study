# mode方法

返回给定键的众数值。


```php
collect([1, 1, 2, 4])->mode();

/**
=> [
     1,
   ]
 */
```

```php
collect([1, 1, 2, 2])->mode();

/**
=> [
     1,
     2,
   ]
 */
```

传入指定键

```php
collect([['foo' => 10], ['foo' => 10], ['foo' => 20], ['foo' => 40]])->mode(
  'foo'
);

/**
=> [
     10,
   ]
 */
```
