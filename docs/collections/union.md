# union

将给定的数组添加到集合中。如果给定的数组中含有与原集合一样的键，则原集合的值不会被改变。

```php
collect([1 => ['a'], 2 => ['b']])->union([3 => ['c'], 1 => ['b']]);

/**
=> Illuminate\Support\Collection {#1087
     all: [
       1 => [
         "a",
       ],
       2 => [
         "b",
       ],
       3 => [
         "c",
       ],
     ],
   }
 */
```
