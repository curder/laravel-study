数据库 Database

Laravel 的 Eloquent ORM 提供数据库抽象，毫不费力地查询和更新数据库数据。Eloquent 与 MySQL、Postgres、SQLite 和 SQL Server 可以完美搭配

队列 Queues

将缓慢的任务放到后台执行，例如发送电子邮件和生成报告，能保持极快的响应时间。支持使用 Redis、Amazon SQS 甚至 MySQL 和 Postgres 作为驱动处理任务

WebSockets

Laravel Echo 和事件广播让构建现代、实时的用户体验变得轻而易举。创建惊人的实时应用程序，同时使用纯 PHP、Node.js 或无服务器解决方案（如 Pusher 和 Ably）为您的 WebSockets 提供支持

权限 Authentication

Laravel 为基于会话的安全身份验证提供脚手架，而 Laravel Sanctum 为 API 和移动应用程序提供无痛身份验证
